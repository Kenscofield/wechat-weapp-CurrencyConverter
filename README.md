## 功能点：
* view、text等标签的基本运用
* wxss的基本运用
* this的指向运用
* 列表式渲染数据
* 带参数页面跳转
* picker标签的使用
* 自定义函数和调取
* wx.request()请求后台数据的运用
* 异步请求多个数据的运用
* 等等
